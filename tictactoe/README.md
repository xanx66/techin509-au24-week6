# Tic Tac Toe Game

This is a tic tac toe game playable in the terminal

# How to play
1. Make sure you have Python installed

# Setup Environment
'''
git clone https://github.com/xanx66/techin509-au24-week6.git

cd techin509-au24-week6/tictactoe

python3 -m venv venv
